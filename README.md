# マークダウン記法　メモ
<http://codechord.com/2012/01/readme-markdown/>

## 段落
空行を開ける

##改行
行末に2つスペース

##強調
アスタリスクまたはアンダーバーで囲む

##コード表示
バッククォーテーションで囲む

##リスト
行頭にアスタリスク or 番号ピリオド（）
1. ほげ
2. ふが

##見出し
行頭の＃の数でH1からH6を表す

##引用
＞を使う。途中で開業してもダブルクォーテーション内は改行されない。

##水平線
3つ以上のハイフン、アスタリスク、アンダースコアを並べる。
